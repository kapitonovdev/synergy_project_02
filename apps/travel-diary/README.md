# Travel Diary Practice

Приложение «Дневник путешествий» для кейс-задачи № 5.

## Функции

- создание пользователя смоделировано переключателем демо-профиля;
- запись собственного путешествия;
- просмотр путешествий других пользователей;
- местоположение путешествия;
- изображение места в виде визуального блока;
- стоимость путешествия;
- места для посещения;
- оценка безопасности.

## Запуск

```bash
pnpm install
pnpm --dir apps/travel-diary dev
```

Для подключения постоянного хранения:

```bash
cd apps/travel-diary
cp .env.example .env
pnpm prisma:migrate
```

`.env`:

```env
DATABASE_URL="file:./dev.db"
```

## Анализ выполнения

1. Основной сценарий - создание и чтение записей о путешествиях.
2. Пользователь может отделить свои записи от общей ленты.
3. Местоположение помогает структурировать путешествия географически.
4. Изображение места добавляет визуальную идентификацию маршрута.
5. Стоимость позволяет сравнивать поездки по бюджету.
6. Список мест для посещения превращает запись в практический маршрут.
7. Prisma-схема готовит приложение к хранению пользователей, поездок и мест.

## Рекомендации по устранению ошибок

1. Добавить загрузку настоящих изображений.
2. Подключить геокодирование и карту.
3. Добавить серверную авторизацию.
4. Добавить приватные и публичные записи.
5. Реализовать редактирование и удаление путешествий.
6. Добавить фильтр по городу и стоимости.
7. Покрыть форму создания записи тестами.
